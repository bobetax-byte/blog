Markdown 历史

Markdown是一种轻量化的标记语言，可以使用纯文本格式来编写文档。2004年由英国人John Gruber 创建。可导出为多种格式。

Markdown 列表

Markdown 的列表有无序列表和有序列表两种情况。

  • 无序列表
    采用 “*” 、”+” 或者 “-“ 作为标记，eg:
  • 有序列表
    有序列表使用序号加上 “.” 来标识
  • 列表嵌套
    列表嵌套只需在子列表中的选项前面添加四个空格即可
